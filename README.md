# discordbot

Just a simple Discord bot

## Getting Started

### Prerequisites
1. Node

### Installing

1. Create a `.env` file on the root directory and place your token value `TOKEN=[TOKEN HERE]`
2. Use the correct Node version `nvm use`
3. Install all dependencies `npm run install`
4. Build packages `npm run build`
5. Start the bot `npm run start`
