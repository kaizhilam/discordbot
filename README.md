# discordbot

Just a simple Discord bot

## Getting Started

### Prerequisites
1. Node

### Installing

1. Create a `.env` file on the root directory and place your token value `TOKEN=[TOKEN HERE]`
2. Install all dependencies `npm install`
3. Build packages `npm run build`
4. Start the bot `npm run start`

### Running the tests

1. Start the test `npm run test`
2. Coverage should be in `coverage\lcov-report\index.html`
